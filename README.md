# 🚀 TechNova Digital Solutions

A modern, responsive, and visually engaging multi-page business website built using HTML5, CSS3, and JavaScript. The project showcases a professional digital agency with elegant UI, smooth animations, and a fully responsive layout.

---

🔗 https://simranjaiswal801.github.io/TechNova/

## ✨ Features

- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- 🚀 Attractive Hero Section
- 👨‍💻 About Us Page
- 💼 Services Page
- 💰 Pricing Plans
- ⭐ Testimonials
- 📊 Business Statistics
- 👥 Team Section
- 📞 Contact Form

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/TechNova-Digital-Solutions.git
```

Open the project folder

```bash
cd TechNova-Digital-Solutions
```

Run

Simply open **index.html** in your browser.
